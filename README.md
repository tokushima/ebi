ebi 
====
__2012-12-25__

Web Framework (PHP 5 >= 5.4.0)

![](ebi.png)


### composer install

```
$ curl -s http://getcomposer.org/installer | php
```

### ebi install

```
$ php composer.phar require tokushima/ebi dev-master
```


### cmdman install

```
$ curl -Lo http://git.io/cmdman.phar
```

### create start file

```
$ php cmdman.phar ebi.Dt::setup --create
```
